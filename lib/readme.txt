Compiere2.exe
-------------
Requires that the following environment variables are set:
- JAVA_HOME
- COMPIERE_HOME

Alternatively, you could start Compiere via utils/RUN_Compiere2


oracle.jar
----------
- Oracle JDBC 9.2 (ojdbc14)


jPDF.jar
---------------
- PDF Printer and Viewer from Qoppa Software (demo)
  The license is included in ComPiere Support.
