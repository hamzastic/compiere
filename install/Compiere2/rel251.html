Migration

From 2.5.0 to 2.5.1 there are about 1200 database structure and 20,000 data changes. On a medium server, it takes about 13 minutes for the structure changes and 10-15 minutes for the data synchronization for a small database. For larger databases, the post-scripts of the data synchronization will take longer.

Installation Requirement

Before installing Release 2.5.1, you MUST delete the jboss sub-directory (e.g. Compiere2/jboss)! This is required as name of the configuration files changed.

Pre Migration to R2.5.1f

Due to Server Infrastructure changes:

  • Delete the jboss sub-directory before upgrading
Post Migration
  • After upgrading, you need to define an
    - Accounting Processor and
    - Request Processors
    for every client (in the Admin Role).
    They are created automatically if you create a new client, but not for existing ones.
  • After you migrated, start the Compiere Client on the server to complete post migration steps.
